# API RESTful de bolsa de trabajo
<h2>Instalacion:</h2>
<p>Instalar dependencias</p>
<pre>
    <code>npm install</code>
</pre>
<p>Crear archivo .env con el siguiente codigo:</p>
<pre>
    <code>
        PG_HOST=127.0.0.1
        PG_USER=
        PG_DATABASE=sbtdb
        PG_PASSWORD=
        PG_PORT=5432
    </code>
</pre>
<p>Correr servidor en modo de desarrollo</p>
<pre>
    <code>
        npm run dev
    </code>
</pre>
