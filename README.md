# API RESTful de bolsa de trabajo
<h1>Instalacion:</h1>
<p>Instalar dependencias -> npm install</p>
<p>Crear archivo .env con el siguiente codigo:</p>
<pre>
    <code>
        PG_HOST=127.0.0.1
        PG_USER=
        PG_DATABASE=sbtdb
        PG_PASSWORD=
        PG_PORT=5432
    </code>
</pre>
<p>Correr servidor en modo de desarrollo -> npm run dev</p>
